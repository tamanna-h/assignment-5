Question-1: What are the different ways to select an element in the DOM?

There are 5 ways in which can select elements in a DOM using selectors.

  • getElementsByTagName()
  • getElementsByClassName()
  • getElementById()
  • querySelector()
  • querySelectorAll()
Question-2: What is the difference between innerHTML, innerText, and textContent ?
InnerHTML returns text of an element with spacing and tags, innerText returns element text without spacing and tags and textContent returns element text with spacing but no tags
Question-3: What is event delegation in the DOM?
Event delegation is a technique in JavaScript where you attach a single event listener to a parent element, rather than multiple listeners to each child. The parent listens for events that bubble up from its children.
Question-4: What is event bubbling in the DOM?
Event bubbling is a DOM behavior where an event starts at the most specific element (the target) and then "bubbles up" through its ancestors (like its parent, grandparent, etc.
Question-5: How do you create, add, and remove elements using JavaScript?
  • Creating an element: const element = document.createElement("tagName"),
  • Appending an element: parentElement.appendChild(element),
  • Removing an element: parentElement.removeChild(element)
